# AxiomNova Open Source Font

AxiomNova is a modern sans-serif typeface, regenerated and refined from historical specimens for open source use. This package includes Regular, Bold, Italic, and Bold Italic styles in TTF, OTF, SFD, and webfont formats (WOFF/WOFF2), with kerning and basic OpenType features.

## Features
- Four styles: Regular, Bold, Italic, Bold Italic
- Formats: TTF, OTF, SFD, WOFF, WOFF2
- Cleaned and validated glyphs (Basic Latin)
- Kerning pairs for improved typography
- Open source license (OFL 1.1)
